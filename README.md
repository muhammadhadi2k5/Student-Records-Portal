# AI Campus Assistant
A backend application created during my software development internship at Highnoon Laboratories LTD.

## What it will include:
1. A fully functional student CRUD
2. Authentication
3. An AI chatbot
4. Documentation (setup/run/test)
5. Voice interaction
6. Realtime notifications
7. Fully Functional database (starting from ER-diagram)
8. Socket.IO

## Tech Stack:
Node.js, TypeScript, (Express/Nest later), Git, ESLint, Prettier, Postman

## Project Status:
In active development.
